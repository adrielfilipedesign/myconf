# A shell script for after installing linux

# how to use

### 1- Download files:
$ git clone https://github.com/adriel-filipe/afbox
### 2 - go to folder:
$ cd afbox
### 3 - make script executable:
$ sudo chmod +x scriptname.sh
### 4 - run the script:
$ sudo ./scriptname.sh
